# Real-Time Chat Application

## 📌 Overview
This project is a real-time chat application built using Node.js, Express.js, MongoDB, and Socket.IO. It allows users to send and receive messages instantly and stores chat history in the database.

## 🚀 Features
- User Registration and Login
- Real-time messaging using Socket.IO
- Chat history stored in MongoDB
- Basic API endpoints
- Unit testing using Mocha and Chai

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Socket.IO
- Mocha & Chai
